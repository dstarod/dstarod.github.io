Создание сайта

Установка

brew install hugo

Проверим

hugo version

Создадим сайт

hugo new site my_blog

cd my_blog

Установим тему

cd themes git submodule add https://github.com/luizdepra/hugo-coder themes/coder cd ../

Создадим контент

hugo new content content/hello.md

Запускаем демо-режим

Сборка - даже помеченные как черновик

hugo server –buildDrafts

Прыгать к измененным страницам

hugo server –navigateToChanged –buildDrafts

Просто собрать

hugo
